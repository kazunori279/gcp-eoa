## The fix that mattered: ship a tested reference, deploy once

Run 1's M2 was the workshop's worst module: ~75 min, ~8 deploys, and a deployed agent that
returned empty responses (3 stacked runtime bugs). Two changes fixed it:

1. **Tested reference `agent_runtime_app.py`** (now in the repo, served via Pages). Step 1
   `curl`s it instead of asking the model to hand-write region handling, memory fallback,
   per-request client, and a 4-method `session_id` override. Step 1's prompt dropped from
   **2,613 → 1,046 chars (~60%)**, and the brittle plumbing is now a reviewed, smoke-tested
   file instead of being re-derived every run.
2. **Local smoke test before deploy** (`runtime_smoke.py`, also shipped). It catches the three
   deploy-blockers (import crash, `global` location, event-loop) in seconds — so the single
   deploy succeeds first try instead of being discovered bug-by-bug across ~8 deploys.
